Vulcan's Mind

Локальна сторінка для тестування плаваючого віджета чату. Натисніть кнопку у правому нижньому куті, щоб відкрити панель.

Віджет очікує, що API доступне за адресою з window.VULCANS_MIND_CONFIG.apiBaseUrl. За замовчуванням вона налаштована на http://localhost:3987 — той самий порт, на якому стартує локальний бекенд (див. api/src/index.ts).

Без реального OPENAI_API_KEY та DATABASE_URL бекенд відповідає у mock-режимі — це нормально для перевірки UX.
