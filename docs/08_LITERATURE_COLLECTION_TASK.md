# Vulcan’s Mind — Literature Collection Task

## Task

Collect and curate 35–50 high-quality sources across 8 topics for the RAG KB. Do not exceed 60 approved P0 sources.

## Output files

- `knowledge/sources.seed.json`
- `knowledge/sources.rejected.json`
- `knowledge/sources.deferred.json`
- `knowledge/notes/source_selection_summary.md`
- project notes in `knowledge/notes/volodymyr_projects_context/`

## Search targets

AI automation: LLM workflow automation, human-in-the-loop, automation reliability.
RAG: foundational RAG, recent surveys, hallucination/grounding, evaluation, vector retrieval.
Psychometrics: PHQ-4, PSS-4, short screening, EMA, validity, screening vs diagnosis.
Wartime mental health: PTSD, moral injury, anxiety/depression, veterans, Ukraine/conflict evidence.
Military medicine: combat casualty care, medevac constraints, austere environments, human factors.
Neuromodulation: rTMS/TMS, tDCS/tACS, neurofeedback, VNS, wearables, sleep/circadian evidence.
Cognitive: cognitive load, adaptive learning, sleep/stress/performance, executive function.
HCI: trust calibration, personalization, user modeling, adaptive interfaces, voice accessibility.

## Critical rules

No restricted full-text PDFs, no sensitive military info, no personal medical data, no marketing claims as evidence, no runtime web search, no overcollection.

## Final report

Approved count by topic, rejected/deferred count, weak areas, copyright limitations, ingestion readiness.
