# Vulcan’s Mind — Knowledge Base Plan

## Goal

Create a curated KB of 35–50 approved sources, maximum 60, covering 8 topics.

## Target distribution

AI automation 4–6; RAG/LLM/chatbots 5–7; psychometrics 5–7; wartime mental health 5–7; military medicine 4–6; neuromodulation 6–8; cognitive performance 4–6; human-AI interaction 4–6.

## Quality hierarchy

Prefer guidelines, systematic reviews, meta-analyses, high-quality reviews, foundational papers, recent empirical papers, official docs, original project notes.

Avoid random blogs, marketing pages, predatory journals, sensational content, unsupported medical claims, sources implying screening equals diagnosis.

## Source schema

Fields: id, title, authors, year, topic, sourceType, url, doi, abstract/summary, publisher, language, status, relevanceScore, qualityNotes, copyrightStatus.

## Copyright

Do not commit restricted PDFs/paywalled full text. Store metadata, DOI/URL, short original notes, abstracts where allowed, open-access full text only when redistributable.

## Required project notes

`vulcans_mind_project_note.md`, `lusterko_project_note.md`, `imenuy_mene_project_note.md`; all factual, modest, public, no sensitive details, no inflated claims.
